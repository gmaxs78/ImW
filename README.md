# ImW

Public release repository for the `ImWatch` VS Code extension.

## Install

Download and install the VSIX:

- [imwatch-0.0.1.vsix](https://github.com/gmaxs78/ImW/raw/main/imwatch-0.0.1.vsix)

In VS Code:

1. Extensions
2. `...`
3. `Install from VSIX...`
4. select `imwatch-0.0.1.vsix`

## Update Manifest

This repo also hosts the public update manifest used by the extension:

- [update.json](https://raw.githubusercontent.com/gmaxs78/ImW/main/update.json)

Default manifest URL inside the extension:

```text
https://raw.githubusercontent.com/gmaxs78/ImW/main/update.json
```
